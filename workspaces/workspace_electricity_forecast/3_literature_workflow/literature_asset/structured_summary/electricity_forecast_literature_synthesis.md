# 电力尖峰预测文献综合笔记

## 1. 当前文献底稿范围
- 已迁移的结构化摘要、爬取结果和示例综述构成了当前主题的初始文献底稿。
- 现有材料主要覆盖 MTGNN、TimesNet、XGBoost、时空序列预测、负荷预测与特征工程等方向。

## 2. 主题脉络
- MTGNN 适合建模电力负荷的空间关联与图结构依赖。
- TimesNet 更适合捕捉多尺度时间模式，对峰值波动和周期性尖峰有潜在优势。
- XGBoost 在结构化特征工程和极端负荷场景中具有强基线价值。
- PatchTST 可作为长窗口时间建模补充，用于尖峰前后的上下文提取。

## 3. 可延展的研究缺口
- 尖峰样本稀少，普通 MAE 很难反映峰值质量，需要峰值专用指标。
- 不同模型擅长的窗口尺度不同，单模型在峰值预测上容易失衡。
- 负荷极端值对特征工程和采样策略较敏感，建议引入尖峰窗口与分位数特征。

## 4. 当前工作建议
- 以尖峰召回率、peak_MAE 与普通 MAE 共同作为评估轴。
- 按 MTGNN / TimesNet / XGBoost / PatchTST 四类方法建立对照实验。
- 优先围绕“尖峰感知”和“极端样本增强”设计创新点。
