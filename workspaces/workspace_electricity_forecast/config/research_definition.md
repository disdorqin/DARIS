# DARIS 研究问题定义 - 电力尖峰预测研究

## 1. 核心主题
基于 MTGNN/TimesNet/XGBoost/PatchTST 的电力尖峰预测研究

## 2. 关键词

### 核心关键词
- 电力尖峰预测
- 尖峰负荷预测
- peak load forecasting
- MTGNN 尖峰预测
- TimesNet 尖峰预测
- XGBoost 尖峰预测
- PatchTST 尖峰预测
- spatiotemporal peak forecasting

### 扩展关键词
- 短期尖峰预测
- 电力尖峰识别
- 负荷极值特征工程
- 极端负荷预测
- 尖峰负荷特性分析

### 排除类型
- 综述
- 长期负荷预测
- 纯理论无实验
- 教学案例

## 3. 文献规则
- **时间范围**：2021-2026 年
- **引用要求**：≥10 次
- **来源要求**：顶刊 + 核心期刊
- **最大数量**：20 篇
- **请求间隔**：5 秒
- **去重处理**：自动去重

## 4. 评价指标

### 主指标
- **val_MAE**：越低越好
- **peak_MAE**：尖峰时段误差越低越好

### 辅助指标
- **peak_recall**：尖峰召回率越高越好
- **peak_precision**：尖峰精度越高越好
- **val_R²**：越高越好
- **推理速度**：每秒样本数
- **模型参数量**：百万参数

## 5. 实验约束
- 使用 MTGNN / TimesNet / XGBoost / PatchTST 四类模型
- 单轮训练时长 ≤ 30min
- 模型参数量 ≤ 基线 2 倍
- 禁止修改数据预处理逻辑
- 显存占用 ≤ 80%
- 每轮必须记录普通误差与尖峰误差

## 6. 迭代规则
- 最大迭代轮次：50 轮
- 指标达标可提前终止
- 支持 7*24 无人值守运行
- 每轮迭代自动 Git 提交
