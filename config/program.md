# MTGNN/TimesNet/XGBoost 调优规则

## 1. 优化目标
提升三模型时序预测精度，主指标 val_MAE 最小化

## 2. 模型调优方向

### MTGNN
- **可调参数**：
  - 图卷积 K 值
  - 隐藏层维度
  - 学习率
  - 权重衰减
- **禁止修改**：
  - 节点数
  - 时序窗口
  - 数据预处理逻辑

### TimesNet
- **可调参数**：
  - d_model
  - num_blocks
  - top_k
  - dropout
  - 学习率
- **禁止修改**：
  - 时序分解逻辑
  - 预测窗口

### XGBoost
- **可调参数**：
  - max_depth
  - learning_rate
  - n_estimators
  - subsample
- **允许操作**：
  - 时序特征组合
  - SHAP 特征筛选
- **禁止修改**：
  - 目标列
  - 数据集划分

## 3. 迭代规则
- 每次修改 1-2 个参数
- 运行完整训练
- 指标优化则保留，否则回滚
- 总迭代 50 轮
- 达标提前终止

## 4. 异常处理规则

### 代码修改异常
- 修改失败：回滚上一版本，记录原因
- 语法错误：自动修复，最多 5 次
- 运行报错：分析日志，Aider 修复

### 指标异常
- 指标下降：回滚最优版本
- 连续 5 轮下降：调整探索方向
- 指标 NaN：检查数据，重新初始化

### 超时处理
- 单轮训练超时：终止任务，调整参数
- 重试 2 次后暂停

## 5. 实验约束
- 单轮训练时长 ≤ 30min
- 模型参数量 ≤ 基线 2 倍
- 显存占用 ≤ 80%
