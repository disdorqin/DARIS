# MTGNN/TimesNet/XGBoost 调优规则

## 1. 优化目标
提升三模型时序预测精度，主指标 val_MAE 最小化

## 2. 模型调优方向

### MTGNN
- **可调参数**：
  - 图卷积 K 值
  - 隐藏层维度
  - 学习率
  - 权重衰减
- **禁止修改**：
  - 节点数
  - 时序窗口
  - 数据预处理逻辑

### TimesNet
- **可调参数**：
  - d_model
  - num_blocks
  - top_k
  - dropout
  - 学习率
- **禁止修改**：
  - 时序分解逻辑
  - 预测窗口

### XGBoost
- **可调参数**：
  - max_depth
  - learning_rate
  - n_estimators
  - subsample
- **允许操作**：
  - 时序特征组合
  - SHAP 特征筛选
- **禁止修改**：
  - 目标列
  - 数据集划分

## 3. 迭代规则
- 每次修改 1-2 个参数
- 运行完整训练
- 指标优化则保留，否则回滚
- 总迭代 50 轮
- 达标提前终止

## 4. 异常处理规则

### 代码修改异常
- 修改失败：回滚上一版本，记录原因
- 语法错误：自动修复，最多 5 次
- 运行报错：分析日志，Aider 修复

### 指标异常
- 指标下降：回滚最优版本
- 连续 5 轮下降：调整探索方向
- 指标 NaN：检查数据，重新初始化

### 超时处理
- 单轮训练超时：终止任务，调整参数
- 重试 2 次后暂停

## 5. 实验约束
- 单轮训练时长 ≤ 30min
- 模型参数量 ≤ 基线 2 倍
- 显存占用 ≤ 80%

## 6. 成本与资源约束（v3.0 新增）

### API 调用成本
- 单次文献检索成本上限：$0.50
- 单日 API 调用总成本上限：$5.00
- 单项目总 API 成本上限：$50.00

### 计算资源约束
- GPU 显存占用 ≤ 80%
- CPU 使用率 ≤ 90%
- 内存使用率 ≤ 85%
- 磁盘空间剩余 ≥ 10GB

### 阿里云资源约束
- 单实例最大运行时长：24 小时
- 单项目总 GPU 时长上限：100 小时
- 自动释放空闲实例

## 7. 早停策略（v3.0 新增）

### 指标早停
- 连续 10 轮指标无改善（变化 < 1%），触发早停
- 达到预设目标指标，触发早停
- 指标持续下降超过 5 轮，触发早停并回滚

### 资源早停
- 显存占用持续 > 90% 超过 5 分钟，触发早停
- 训练时长超过预设上限，触发早停
- API 成本达到上限，触发早停

### 异常早停
- 连续 3 次实验运行失败，触发早停
- 检测到数据异常（NaN/Inf），触发早停
- 系统资源不足，触发早停

### 早停后处理
- 保存当前最优模型
- 记录早停原因与状态
- 生成早停报告
- 通知研究管理智能体
